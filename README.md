# Retail Sales Analysis & Prediction

## Project Overview

This project analyzes retail sales data and predicts total sales using Linear Regression.

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed

- Data Exploration
- Sales Trend Analysis
- Category-wise Analysis
- Region-wise Analysis
- Correlation Analysis
- Sales Prediction

## Machine Learning

- Linear Regression

## Evaluation

- Mean Absolute Error (MAE)
- R² Score

## Conclusion

This project demonstrates an end-to-end retail data analysis and prediction workflow using Python.
